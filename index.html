3D-Barolo (3D-Based Analysis of Rotating Object via Line Observations) or BBarolo is a tool for fitting 3D tilted-ring models to emission-line datacubes.

BBarolo 1.4 is coming soon, with new exciting features (including pyBBarolo, a BBarolo python package!).

In the meanwhile, check the current version out (BBarolo 1.3).
